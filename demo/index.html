For website developers

Add an "Open in Agent" button to any page in 30 seconds.

Option A: Link (works with extension installed)
<a href="agent://read?url=https://your-site.com/docs">
  Open in Agent
</a>
Option B: Fallback-safe (works for everyone)
<a href="https://agentopen.dev/send?url=https://your-site.com/docs&prompt=Read+this+and+follow+instructions">
  Open in Agent
</a>
Option C: JavaScript button (richest UX)
<button class="agent-btn"
  data-agent-action="read"
  data-agent-url="https://your-site.com/docs">
  Open in Agent
</button>
<script src="https://agentopen.dev/sdk.js"></script>

The agent:// URI Spec

A proposed standard for agent-addressable actions on the web.

URI Action Description
agent://read?url=... Read Send a URL for the agent to read and understand
agent://execute?url=... Execute Send instructions the agent should follow step-by-step
agent://install?skill=... Install Install a skill/plugin from a SKILL.md URL
agent://chat?prompt=... Chat Start a conversation with a pre-filled prompt
Full URI format
agent://<action>?url=<encoded_url>&prompt=<encoded_prompt>&harness=<preferred_harness>

Examples:
agent://read?url=https://docs.stripe.com/api/charges
agent://execute?url=https://example.com/skill.md
agent://install?skill=https://skills.sh/my-skill/SKILL.md
agent://chat?prompt=Explain%20quantum%20computing
