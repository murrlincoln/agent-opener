Deploy to Cloudflare Workers SKILL

A step-by-step skill for deploying serverless functions to Cloudflare Workers with wrangler.

Prerequisites

  • Node.js 18+ installed
  • Cloudflare account with Workers enabled
  • Wrangler CLI: npm install -g wrangler

Steps

1. Authenticate with Cloudflare

wrangler login

2. Initialize a new Worker project

wrangler init my-worker
cd my-worker

3. Write your handler

export default {
  async fetch(request) {
    return new Response("Hello from Workers!");
  },
};

4. Test locally

wrangler dev

5. Deploy to production

wrangler deploy

Verification

After deploying, wrangler prints the live URL. Curl it to confirm:

curl https://my-worker.your-subdomain.workers.dev
